# ARCP API Example Files

These examples are intentionally split by concern so each file stays focused.

- `ArcpSignalExamples`: publishing values, arrays, metadata, topic handles
- `ArcpControlExamples`: writable fields, command invoke wiring, event dispatch
- `ArcpReadExamples`: optional cached reads and metadata snapshots
- `ArcpLayoutExamples`: layout DSL and typed device widget layout JSON
- `ArcpRobotCoreExamples`: RobotCore builder/runtime integration for ARCP
